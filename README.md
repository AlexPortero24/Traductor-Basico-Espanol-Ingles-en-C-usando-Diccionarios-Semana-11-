Traductor Español → Inglés en C#

## 📌 Descripción

Aplicación de consola desarrollada en C# que permite traducir frases del español al inglés utilizando un diccionario interno. 

El programa traduce únicamente las palabras registradas en el diccionario y mantiene sin cambios aquellas que no existen.

---

## 🎯 Objetivo

Desarrollar una aplicación que implemente la estructura `Dictionary` para almacenar palabras en español y sus equivalencias en inglés, permitiendo la traducción de frases completas.

---

## ⚙️ Funcionalidades

- Traducción de frases completas (traducción parcial).
- Diccionario inicial con varias palabras (español → inglés).
- Opción para agregar nuevas palabras.
- Menú interactivo en consola.
- Procesamiento básico de texto.
